HTML Forms

Input Type = Text

This html input element accepts text input from user. The default attribute "type" is text. Id and name attributes commonly used with the same values

Input Type = Text with Label

Same with example above but with label that have corresponding "for" attribute. The "for" attribute transfers the focus to the input when the label is clicked.

Input Type = Number with Label

In this html input, it accepts numbers only. Minimum and maximum values can be restricted with "min" and "max" attributes.

Input Type = Range with Label

Range input also accepts numbers only as input but not as a textbox instead it presents as slider to get the value from user. It has "step" attribute to restrict the step between values.
